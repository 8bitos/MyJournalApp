Bantuan Penggunaan MyJournal

1. Login dan inisialisasi

Klik Masuk dengan Google. Sistem akan meminta izin akses Google Drive dan Google Sheets. Setelah berhasil, aplikasi akan membuat (jika belum ada):

- Folder gambar jurnal di Google Drive.

- Spreadsheet rekap jurnal bulanan.

Jika izin ditolak, aplikasi tidak dapat digunakan karena semua data bergantung pada Google API user.

2. Menambah catatan

Klik Tambah Catatan, isi field yang tersedia, upload foto (opsional/mandatory sesuai setting), lalu klik simpan.

Data akan disimpan sebagai 2 bagian:

- Teks/form input ke Google Sheets.

- File gambar ke Google Drive.

Bulan penyimpanan ditentukan dari field bertipe date. Contoh tanggal 2026-04-10 masuk ke sheet 2026-04.

3. History bulanan

Gunakan dropdown bulan di toolbar untuk berpindah riwayat. Setiap bulan dipisah dalam tab sheet format YYYY-MM agar rekap lebih rapi dan cepat ditelusuri.

Perpindahan bulan hanya memfilter tampilan dan sumber data aktif, bukan memindahkan data otomatis kecuali saat edit tanggal catatan.

Untuk mode grup/sekolah, gunakan filter Guru untuk melihat catatan per guru atau seluruh guru pada bulan tersebut.

4. Edit dan hapus

Gunakan ikon edit/hapus pada kartu jurnal.

- Edit: memperbarui data baris aktif. Jika tanggal berubah ke bulan lain, data otomatis dipindah ke sheet target bulan.

- Hapus: menghapus baris di Sheets dan mencoba menghapus gambar terkait di Drive.

5. Export data bulanan

Klik Export Bulan untuk mengunduh data dari bulan yang sedang aktif dalam format CSV.

Langkah ekspor:

1. Pilih bulan pada dropdown Riwayat.

2. Klik tombol Export Bulan.

3. File otomatis terunduh dengan nama journal-YYYY-MM.csv.

4. Buka file CSV dengan Excel atau upload kembali ke Google Sheets jika diperlukan.

6. Setting Input

Gunakan Setting Input untuk mengatur struktur form tambah catatan sesuai kebutuhan kerja Anda.

Fungsi setiap kontrol di Setting Input:

- Label: nama field yang tampil di form dan kartu jurnal.

- Type: jenis input field.

- Wajib: jika dicentang, field harus diisi sebelum simpan.

- Hapus: menghapus field dari form aktif.

- Tambah Field: menambah field baru.

- Reset Default: mengembalikan struktur ke format awal sistem.

Jenis type yang tersedia:

- text: input teks singkat (contoh: Nama Siswa, Bidang Layanan).

- textarea: input teks panjang (contoh: Catatan, Evaluasi).

- date: tanggal. Minimal harus ada 1 field date agar pembagian bulanan berjalan.

- time: jam kegiatan.

- select: dropdown pilihan. Jika memilih type ini, isi Opsi dropdown dipisahkan koma.

Contoh opsi select: Kelas 7.1, Kelas 7.2, Kelas 7.3.

Setting input disimpan per akun Google di sheet konfigurasi, sehingga user lain tidak ikut terpengaruh walau memakai spreadsheet yang sama.

7. Reset ke default

Di menu setting, klik Reset Default lalu simpan pengaturan untuk kembali ke struktur awal.

8. Dark Mode

Klik tombol Dark Mode di toolbar untuk ganti tema terang/gelap. Preferensi disimpan otomatis di browser.

9. Mode Organisasi (opsional)

Tombol Organisasi ada di menu toolbar. Secara default aplikasi tetap di mode personal.

Mode ini dipakai jika beberapa guru/anggota ingin menulis pada spreadsheet dan folder gambar yang sama.

Konsep dasar:

- Personal: pakai spreadsheet + folder milik akun sendiri.

- Organisasi: pakai spreadsheet + folder milik organisasi (owner).

Di modal Organisasi tersedia aksi:

1. Buat Organisasi: membuat resource organisasi baru (spreadsheet + folder images).

2. Join Organisasi: gabung memakai token invite dari owner.

3. Kelola Organisasi: owner dapat share akses editor dan membuat invite.

Tombol pada Kelola Organisasi:

- Share + Copy Invite: validasi email, share akses editor ke spreadsheet + folder, lalu copy token invite.

- Copy Invite Token: menyalin token invite tanpa proses share email.

- Simpan Organisasi: menyimpan organisasi aktif ke daftar lokal akun/browser ini.

- Hapus Organisasi: menghapus spreadsheet + folder organisasi (khusus host/owner).

- Kembali ke Personal: keluar dari mode organisasi dan kembali ke resource pribadi.

- Masuk Lagi Terakhir: aktifkan ulang organisasi terakhir tanpa paste token ulang.

Catatan penting:

- Anggota wajib sudah diberi akses oleh owner agar bisa menulis data.

- Jika hanya dapat token tanpa akses folder/sheet, user bisa gagal simpan.

- Di mode organisasi, sistem menambahkan field Nama Guru agar rekap per guru lebih jelas.

Syarat & Ketentuan

1. Data dan file disimpan pada akun Google pengguna yang login. Aplikasi tidak menyediakan penyimpanan server terpisah.

2. Kuota penyimpanan (Google Drive) sepenuhnya mengikuti kapasitas akun pengguna. Jika kuota penuh, upload gambar/data dapat gagal.

3. Jika Google Drive/Google Sheets API sedang bermasalah, dibatasi, atau token login kedaluwarsa, proses simpan/edit/export dapat terganggu.

4. Pengguna bertanggung jawab atas isi data yang disimpan, termasuk keamanan akun Google dan pengelolaan akses file.

5. Struktur input kustom disimpan per akun di spreadsheet konfigurasi; perubahan tidak akan diterapkan ke akun Google lain.

6. Untuk menghindari kehilangan data, pengguna disarankan melakukan backup berkala dari spreadsheet dan folder gambar.

7. Lokasi penyimpanan file:

- Gambar tersimpan di folder Drive: My Monthly Journal - Images.

- Data tabel tersimpan di spreadsheet: My Monthly Journal - Data dengan tab bulanan YYYY-MM.

- Konfigurasi setting input akun tersimpan di tab: _CONFIG.

8. Untuk penggunaan banyak guru BK dalam satu sekolah, gunakan fitur Organisasi (buat/join via invitation) agar semua anggota bekerja di spreadsheet dan folder organisasi yang sama.

9. Owner organisasi dapat memakai tombol Share + Copy Invite untuk memberi akses editor ke email anggota sekaligus menyalin token invite.

10. Perubahan setting input tetap bersifat per akun Google, termasuk ketika user sedang berada di mode organisasi.
